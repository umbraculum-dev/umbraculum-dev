Support & sponsorship

Umbraculum is community-first in what gets built and honest about capacity (architecture, maintainer time, AI compute). At public alpha this is a solo-maintained project — collaboration is the highest-leverage ask; financial and in-kind sponsorship are optional.

Collaborate (highest leverage)

Code, reviews, documentation, tests, accessibility work, and execution plans in the umbraculum-dev monorepo (platform, canonical modules, verticals) compound for every future contributor. The open-source umbraculum-toolset Cursor plugin pack is a sister repo — rules, skills, and subagents that keep AI-assisted contributions aligned with the same CI bar.

Start with Getting started, community forum, and CONTRIBUTING.md.

Financial sponsorship (optional)

Help carry AI inference, maintainer time, and infrastructure so the project can ship more of what the community has already voted on — not what a sponsor privately prefers.

At public alpha, operational maintainability — keeping the forum, demo host, and public surfaces reachable — runs on very modest fixed infrastructure: about €15/month today. The lights-on burden stays small; keeping the project running is realistically affordable without jackpot-scale funding. Developer tooling and maintainer time are separate from that figure and are not itemized here yet.

Fixed infrastructure costs today:

  • Community forum (forum.umbraculum.dev) — ~€10/month (VPS, backups, off-site archive)
  • Demo host (demo.umbraculum.dev) — €5/month

Recurring — Liberapay One-time — Buy Me a Coffee

  • Liberapay — weekly/monthly/yearly support; 0% platform fee (Stripe processing only). Best for ongoing sponsorship. PayPal is not used at alpha.
  • Buy Me a Coffee — one-time tip; ~5% platform fee. Liberapay does not support one-off donations.

Phase 0 channels. Adding payment rails (e.g. GitHub Sponsors, Open Collective) requires a community proposal and vote when documented triggers fire — see Core development and community §5.

In-kind — compute, API usage, infrastructure

Payment buttons move cash only. There is no standard “donate API tokens” checkout for open-source projects.

If you want to sponsor AI inference or Cursor/API usage:

  • Contribute cash via Liberapay or Buy Me a Coffee (above), or
  • Arrange organizational billing (your org pays the vendor directly — e.g. Cursor Team, Anthropic/OpenAI org account) or help with a cloud credit program application — coordinate on the forum.

We do not accept shared API keys or credentials by email or chat. Earmarks (“for inference costs”) are welcome as information; they do not buy queue priority (§5.3).

What sponsorship does not buy

  • Vote weight or queue priority
  • Veto immunity or private decision channels
  • Feature paywalls — AGPLv3 core stays available to everyone on the same terms

Sponsorship can buy:

  • Public acknowledgement (with opt-out) — flat alphabetical list on this page; recognition only, no tiers or logo deals. Say if you prefer anonymity.
  • The same visibility as everyone else — public roadmap, RFCs, forum, and community meetings.
  • Helping sustainability — per MANIFESTO §2.1.
